ImageEditPipeline
Automated image processing pipeline with a REST API layer.
screenshot / preview
◤ overview
ImageEditPipeline is a lightweight tool for automating repetitive image tasks. You define a pipeline of operations — resize, convert, watermark, compress — and point it at a folder. It processes everything in batch, with a FastAPI layer on top so it can be triggered remotely or integrated into other workflows. Built because I kept doing the same manual steps in Photoshop every time I exported assets for a project.
